TTOWN is a 7 key macropad with a rotary encoder, and led backlights. It uses KMK firmware to bind keys to your keyboard. The macropad has a grey top with a darker shade for the base to keep a sleek/minimalstic when combined with the led. Which is all held together by 4 heatset inserts and 4 m3x16mm screws.

Keybinds:
Some keybinds include copy, paste, cut, raise and lower volume, and undo/redo.

Firmware: 
I used KMK firmware through circuit python to code my pcb,you can change/control your keybinds and led colors.

BOM:

An EC11 Rotary encoder for scrolling or volume control
1 unsoldered Seeed XIAO RP2040
7x through-hole 1N4148 Diodes
7x MX-Style switches
1x EC11 Rotary encoders
7x white blank DSA keycaps
7x SK6812 MINI-E LEDs
4x M3x16mm screws
4x M3x5mx4mm heatset inserts
